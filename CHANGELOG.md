# Changelog

All notable changes to this Snake Game project will be documented in this file.

## [1.0.0] — 2026-07-06

### Added
- Initial Snake game with canvas rendering
- Arrow key and WASD controls
- Score tracking with localStorage high score
- Level system with speed progression
- Pause/Resume functionality
- Dark theme UI design
- Game configuration file (config.js)
- Responsive game wrapper layout
